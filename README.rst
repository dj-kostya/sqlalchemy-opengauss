Introduction
-------------

本仓库是基于 sqlalchemy commit [fb81f9c](https://github.com/vimiix/sqlalchemy/commit/fb81f9c8d914f9911925dd3f4e77d7fc374b267c)
基础上进行修改，主要新增两个特性：

1. 新增 py_opengauss 驱动支持
2. openGauss 数据库支持多主机连接，自动选主库
